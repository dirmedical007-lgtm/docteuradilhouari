# Cabinet Médical Dr Houari Adil — Pack statique optimisé (GitHub Pages)

Pack **prêt à publier** (HTML/CSS/JS) : rapide, responsive, accessible (skip link, ARIA), SEO local + données structurées.

## Mise en ligne
1. Déposez tous les fichiers à la racine du repo.
2. GitHub → Settings → Pages → Deploy from branch → main → /(root)
3. Votre site sera disponible sur votre domaine / GitHub Pages.
